# Project_Stats
Project Stats Tool (PowerShell)

Just a simple PS script to show project stats

Count total lines across all files<br>
Count total characters across all files<br>
Group and count files by extension<br>
Show the largest files by lines, characters, or size<br>
Show the smallest files by lines, characters, or size<br>
Export full file statistics to CSV<br>
Calculate total project size<br>
List the most common file extensions<br>
Interactive menu that runs automatically

Excluded directories and files (like node_modules, .git, .env, etc.) are filtered automatically.

1. Clone or download the script.
2. Run it in PowerShell: `./project-stats.ps1`
3. When prompted, enter the folder you want to scan.
4. Use the interactive menu to choose what you want to analyse.
